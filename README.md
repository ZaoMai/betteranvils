# BetterAnvils

## 概述
BetterAnvils 是一个 Minecraft Spigot 插件，它重新实现了原版铁砧的功能，
使服务器管理员能够对铁砧的行为进行更加细致的配置，同时也为玩家提供了更加友好的使用体验。

## 功能特性
- 可为每一种附魔单独配置最大附魔等级（例如：时运 VIII、锋利 X 等）
- 可配置附魔与修理时的经验消耗倍率
- 可配置附魔 / 修理的最大经验消耗上限（不再出现 “~~过于昂贵！~~”）
- 可配置不同材料的修理消耗
- 更智能的附魔 / 修理 / 重命名逻辑，防止玩家在无意义的操作上浪费材料、工具或经验等级
